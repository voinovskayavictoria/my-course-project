# API Fuzzer LSTM
Разработка модели для фаззинга (fuzzing) параметров API на основе генерации вероятных вредоносных значений

## 🛠 Требования
* Python 3.9+
* pip

## Как запустить
1. Клонируйте репозиторий:
```
git clone https://github.com/username/repo.git
cd repo
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Запустите проект:
```
python web_app.py
```

Откройте в браузере: http://localhost:5000

Если хотите запустить генерацию из консоли без веб‑интерфейса:
```
python fuzz_generator.py
```
